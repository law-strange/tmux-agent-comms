# docs/

Drop the demo here as `demo.gif` (referenced from the top-level README).

Record it with [asciinema](https://asciinema.org) + [agg](https://github.com/asciinema/agg):

```bash
asciinema rec demo.cast      # show: claude posts to a thread → doorbell → codex reads + replies
agg demo.cast demo.gif
```

Keep it under ~20 seconds: one clear hand-off between two agents.
